Quiz 1:
Design an HTML page and Javascript where: 
1)	you enter a number using a popup generated by JavaScript,
2)	you enter a second number using a popup generated by JavaScript,
3)	calculate a result where the two numbers are multiplied,
4)	display the result in an HTML element with red text,
5)	and design the javaScript as a function. 
Submit necessary files to Brightspace. If embedding JavaScript into your .html file, only submit an .html file. If using JavaScript externally, submit both an .html file and a .js file.
